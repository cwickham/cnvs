# Package index

## Queries

Functions for querying the Canvas LMS API

- [`cnvs()`](https://cwickham.github.io/cnvs/reference/cnvs.md) : Query
  the Canvas LMS API
- [`cnvs_upload()`](https://cwickham.github.io/cnvs/reference/cnvs_upload.md)
  : Upload files to Canvas

## Credentials

- [`cnvs_whoami()`](https://cwickham.github.io/cnvs/reference/cnvs_whoami.md)
  : Info on current Canvas user, domain and token
- [`cnvs_token()`](https://cwickham.github.io/cnvs/reference/cnvs_token.md)
  [`cnvs_token_exists()`](https://cwickham.github.io/cnvs/reference/cnvs_token.md)
  [`cnvs_domain()`](https://cwickham.github.io/cnvs/reference/cnvs_token.md)
  : Return the local user's Canvas token and domain

## Pagination

- [`cnvs_next()`](https://cwickham.github.io/cnvs/reference/cnvs_next.md)
  [`cnvs_prev()`](https://cwickham.github.io/cnvs/reference/cnvs_next.md)
  [`cnvs_first()`](https://cwickham.github.io/cnvs/reference/cnvs_next.md)
  [`cnvs_last()`](https://cwickham.github.io/cnvs/reference/cnvs_next.md)
  : Get the next, previous, first or last page of results

## Utilities

- [`print(`*`<cnvs_response>`*`)`](https://cwickham.github.io/cnvs/reference/print.cnvs_response.md)
  : Print the result of a Canvas API call
